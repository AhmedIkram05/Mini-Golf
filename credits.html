About 2D Mini Golf

2D Mini Golf is a browser-based mini golf game created with JavaScript, HTML, and CSS. This project was developed as a fun way to showcase web development skills and create an enjoyable game experience.

The game features multiple levels, physics-based gameplay, multiplayer mode, and a level editor to create your own courses.
